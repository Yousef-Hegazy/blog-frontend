# Features

## UI Components

- **Beautiful Blog Cards** - Image, title, excerpt, tags, categories
- **Filter Sidebar** - Multi-select categories and tags
- **Search** - Real-time search in header
- **Responsive Grid** - Works on all devices

## Pages

1. **Home** - All posts with filters
2. **Categories** - Browse by category
3. **Tags** - Tag cloud and list

## Filtering

- Multi-select categories
- Multi-select tags
- Real-time search
- Clear all filters
- URL-based filtering

## Tech

- Next.js 16 (App Router)
- React 19
- PrimeReact UI
- Zustand state
- TypeScript
- Tailwind CSS
